# KazSaq_DataHUB

Общий каталог университетов Казахстана с полным функционалом для поиска, сравнения и изучения информации о высших учебных заведениях.

## Возможности

- **Об университете** - информация о миссии, истории, лидерстве и достижениях
- **3D-тур** - виртуальное путешествие по кампусу университета
- **Академические программы** - подробная информация о программах и курсах
- **Международное сотрудничество** - программы обмена, партнерские университеты
- **Приём и поступление** - требования, процедуры, сроки, стипендии
- **Сравнение** - возможность сравнивать университеты и программы
- **ИИ Помощник** - интеллектуальный помощник на базе Groq API для поиска и консультаций

## Технологии

- Flutter 3.7+
- Firebase (Firestore, Storage, Auth)
- Groq API для ИИ функционала
- Provider для управления состоянием
- Кроссплатформенность (Android, Web)

## Настройка проекта

### 1. Установка зависимостей

```bash
flutter pub get
```

### 2. Настройка Firebase

1. Создайте проект в [Firebase Console](https://console.firebase.google.com/)
2. Добавьте приложения для Android и Web
3. Скачайте конфигурационные файлы:
   - `google-services.json` для Android (поместите в `android/app/`)
   - Настройки для Web (скопируйте в `lib/firebase_options.dart`)

4. Обновите `lib/firebase_options.dart` с вашими данными Firebase

### 3. Структура данных Firestore

Создайте коллекцию `universities` в Firestore со следующей структурой:

- `name`, `nameRu`, `nameKz` - названия университета
- `description`, `descriptionRu`, `descriptionKz` - описания
- `logoUrl`, `coverImageUrl` - URL изображений
- `location`, `city` - местоположение
- `website`, `email`, `phone` - контакты
- `rating` - рейтинг
- `studentCount` - количество студентов
- `foundedDate` - дата основания
- `images` - массив URL изображений
- `mission` - объект с миссией
- `history` - объект с историей
- `leadership` - объект с руководством
- `achievements` - массив достижений
- `academicPrograms` - массив программ
- `internationalCooperation` - объект сотрудничества
- `admission` - объект приема
- `tour3dUrl` - URL 3D-тура

### 4. Настройка Groq API (для ИИ помощника)

1. Перейдите на [Groq Console](https://console.groq.com/)
2. Создайте аккаунт или войдите
3. Перейдите в раздел API Keys
4. Создайте новый API ключ
5. В приложении откройте "ИИ Помощник" и введите API ключ через настройки

**Примечание:** API ключ хранится локально в приложении и не передается на серверы.

Подробные инструкции по настройке Firebase и Groq API см. в файле `FIREBASE_SETUP.md`.

### 5. Запуск проекта

```bash
flutter run
```

Для веб-версии:
```bash
flutter run -d chrome
```
